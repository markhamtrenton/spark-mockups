## 🚀 How to Add a New Mockup
1. Create a new folder: `mkdir client-name`
2. Drop the `index.html` mockup inside it
3. Add a card to the main `index.html`
4. Push to GitHub: `git add . && git commit -m "Add [client] mockup" && git push`
5. Share link: `https://YOUR-USERNAME.github.io/spark-mockups/client-name`

## 📱 Pitch Strategy
- Walk in with the link ready on your phone
- Hand them your phone and say *"I already built your site — want to see it?"*
- QR codes inside each demo link directly to DoorDash

## 🔧 Setup (First Time)
```bash
git clone https://github.com/YOUR-USERNAME/spark-mockups
cd spark-mockups
# Make changes, then:
git add .
git commit -m "Your message"
git push origin main
```

Enable GitHub Pages:
- Go to repo Settings → Pages
- Source: Deploy from branch → main → / (root)
- Save — live in ~60 seconds
